# Guidance_System_Module
Development and creation of an autonomous guidance system module with a laser designator for a CubeSat 1U spacecraft, which provides automatic two-axis guidance with a ±40° field of view, radio data reception and transmission, and is compatible with CubeSat standards in terms of dimensions, mounting, and vibration resistance.
Разработанный модуль представляет собой автономную систему наведения с лазерным целеуказателем, предназначенную для установки в космический аппарат форм-фактора CubeSat 1U. Модуль реализует полный цикл управления направлением лазерного луча в двух взаимно перпендикулярных плоскостях с возможностью дистанционного управления по радиоканалу. Основные функциональные возможности включают:
·	Автоматическое двухосевое наведение – лазерный луч исходит из геометрического центра CubeSat; система обеспечивает поворот по двум осям (наклона и поворота) с углом обзора ±40° по каждой оси и шагом позиционирования 10°. Точность наведения проверяется на мишени с расстояния 1 метр.
·	Алгоритм работы устройства – после включения модуль выполняет инициализацию, устанавливается в начальное положение (0°, 0°), переходит в режим ожидания радиокоманд, а при получении управляющего сообщения выполняет заданные последовательности сканирования (горизонтальное, вертикальное и два диагональных) с отправкой данных о текущих углах каждые 3 секунды.
·	Радиосвязь и обмен данными – с использованием радиомодуля; передаваемые сообщения содержат уникальный идентификатор устройства, текущие углы наклона и поворота, состояние режима работы. Время отклика системы не превышает 2-х секунд.
·	Конструкционные особенности – модуль выполнен как самостоятельный блок, совместимый с CubeSat 1U; корпус устойчив к вибрации, изготовлен из пластика 3D-печати.
·	Программно-аппаратная реализация – на базе контроллера Arduino, сервоприводов, лазерного модуля и источника питания. Программное обеспечение реализует алгоритмы управления и обработки команд.
